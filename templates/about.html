About This Project

The motivation, methodology, and technology behind CreditGuard AI.

The Problem

Why credit risk prediction matters

Financial institutions lose billions annually to loan defaults. Traditional rule-based underwriting is slow, inconsistent, and unable to capture complex non-linear patterns in borrower behaviour.

CreditGuard AI was built to solve this — an automated, ML-driven system that evaluates default probability in real-time, reducing human bias and accelerating lending decisions while maintaining regulatory transparency.

Methodology

Stacked ensemble architecture

The core of CreditGuard AI is a Stacking Classifier that combines five diverse base models, each contributing a unique perspective on the data:

01

Logistic Regression

A linear baseline that captures the fundamental decision boundary between default and non-default classes.

02

Decision Tree

Learns interpretable if-then rules from the data, capturing non-linear thresholds in credit features.

03

Random Forest

An ensemble of hundreds of decorrelated trees that reduces variance and improves generalization.

04

XGBoost

Gradient-boosted trees with regularization — highly competitive on tabular data with built-in handling of missing values.

05

CatBoost

Ordered boosting with native categorical feature support, reducing prediction shift and overfitting.

Meta-Learner

A final estimator trained on the base models' cross-validated predictions, learning the optimal way to combine them.

Data

Feature engineering & pipeline

The model was trained on historical credit bureau data with 24 engineered features covering three categories:

Demographic

  • Gender
  • Occupation (19 categories, one-hot encoded)

Employment

  • Number of days employed

Credit History

  • Credit score
  • Credit limit utilization (%)
  • Defaults in last 6 months
Technology

Production stack

XGBoost Gradient boosted tree base model
CatBoost Ordered boosting base model
Flask Lightweight WSGI web framework
Pandas DataFrame construction for inference input
